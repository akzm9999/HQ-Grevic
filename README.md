## Using in webs
 
 #### 1) import
 CSS :
 ```css
 @import url('https://akzm9999.github.io/HQ-Grevic/web/css/HQ-Grevic.css');
 ```
 or
HTML (Recommended) :
 ```html
<link href='https://akzm9999.github.io/HQ-Grevic/web/css/HQ-Grevic.css' rel='stylesheet' type='text/css'>
```

#### 2) apply
- css
```css
font-family: 'HQ Grevic', Sans-serif;
```
